# Heart Disease Prediction using Statistical Analysis

## Overview
This project focuses on analyzing medical data to identify patterns and factors associated with heart disease. Using statistical techniques, the project explores relationships between different health indicators and the likelihood of heart disease, helping to understand potential risk factors.

## Objective
The main objective of this project is to apply statistical methods to analyze heart disease datasets and gain insights that can assist in early prediction and risk assessment.

## Key Features
- Data exploration and preprocessing
- Descriptive statistical analysis
- Correlation analysis between health parameters
- Regression analysis for prediction insights
- Visualization and interpretation of results

## Technologies & Tools Used
- Python  
- Jupyter Notebook / VS Code  
- Statistical analysis techniques  
- Data visualization libraries

## Dataset
The dataset contains various medical attributes such as:
- Age
- Cholesterol levels
- Blood pressure
- Heart rate
- Other relevant health indicators

These attributes are analyzed to determine their relationship with heart disease occurrence.

## Methodology
1. Data collection and loading  
2. Data cleaning and preprocessing  
3. Exploratory data analysis  
4. Statistical testing and correlation analysis  
5. Regression analysis for prediction insights  
6. Result interpretation and conclusion
